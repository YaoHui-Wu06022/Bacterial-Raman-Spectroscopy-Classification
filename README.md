SupCon loss（Supervised Contrastive）

- 作用：在有标签的对比学习中，把同类样本拉近、异类样本拉远，直接优化类内紧凑与类间分离。

- 效果：特征判别性强、对少样本/类内变化鲁棒；常能提升线性可分性与下游分类精度。

- 适用：分类任务、标签可用、希望学到更清晰的类边界。

- 优点：

  类内分布更紧凑

  比传统交叉熵更鲁棒

  适合小样本或长尾问题



Align loss（对齐损失）

- 作用：主要做“类内对齐/收紧”，让同类特征更集中（通常不强力处理类间分离）
- 效果：提高同一语义/同一实体的表示一致性，收敛更稳但类间分离度可能不足，需要配合其他损失
- 适用：多视图/多模态对齐（图文、增强视图）、表示一致性优先的场景

一句话区分

- SupCon：既拉近同类又推远异类，强调判别性
- Align：主要拉近正对，强调一致性，常需额外机制保证分散度

如果两者都太强，确实会出现“过约束”现象（训练更难、泛化下降），实践上常见做法是分阶段启用、并让权重较小

统计池化（Statistical Pooling）

```
mean = out.mean(dim=1)
std = out.std(dim=1, unbiased=False)
feat = torch.cat([mean, std], dim=1)
```

如果类别差异主要体现在：

- 整体形状差异
- 峰强比例差异
- 噪声/粗糙度差异

stat pooling 很稳定

优点：非常稳定、不易过拟合、小样本友好

如果两个类别：

- 平均值相近
- 方差相近
- 但关键峰位置不同

stat pooling 可能分不开

注意力池化（Attention Pooling）

```
att = self.att_pool(out)  # [B, L, 1]
att = torch.softmax(att, dim=1)
feat = (out * att).sum(dim=1)
```

是一个可学习加权平均

模型自己决定哪些波数更重要

优点：

- 可以突出关键波段
- 对微弱但判别性强的峰敏感
- 与 Transformer 结合效果好
- 更高表达能力

风险：

- 小样本容易过拟合
- 注意力可能学到“伪峰”
- 对噪声敏感
- 训练初期不稳定
