SupCon loss（Supervised Contrastive）

- 作用：在有标签的对比学习中，把同类样本拉近、异类样本拉远，直接优化类内紧凑与类间分离。
- 效果：特征判别性强、对少样本/类内变化鲁棒；常能提升线性可分性与下游分类精度。
- 适用：分类任务、标签可用、希望学到更清晰的类边界。

Align loss（对齐损失）

- 作用：强调正样本对在特征空间“对齐”，通常只关注拉近正对，不显式推远负对（或推远较弱）。
- 效果：提高同一语义/同一实体的表示一致性，收敛更稳但类间分离度可能不足，需要配合其他损失（如uniformity或分类损失）。
- 适用：多视图/多模态对齐（图文、增强视图）、表示一致性优先的场景。

一句话区分

- SupCon：既拉近同类又推远异类，强调判别性。
- Align：主要拉近正对，强调一致性，常需额外机制保证分散度。
